Common Errors

All client API errors return JSON. Validation failures include field-level details; auth and scope failures do not expose key internals.

401

Authentication failed

{
  "error": {
    "code": "invalid_api_key",
    "message": "Authentication credentials were not accepted."
  }
}
403

Client scope mismatch

{
  "error": {
    "code": "validation_error",
    "fields": {"client": "forbidden"}
  }
}
405

Wrong method

{
  "error": {
    "code": "method_not_allowed",
    "allowed_methods": ["GET", "POST"]
  }
}
